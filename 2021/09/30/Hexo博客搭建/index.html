Hexo博客搭建

Hexo搭建

下载并安装程序与基础环境

Git下载地址 Node.js载地址

运行 Git Bash 并切换淘宝镜像 (提高下载速度)

全局切换命令：

npm config set registry http://registry.npm.taobao.org/

查看版本命令：

npm get registry

切回官方镜像命令：

npm config set registry http://www.npmjs.org

安装全局hexo依赖

运行命令：

npm install -g hexo

查看版本：

hexo v

创建 hexo 文件夹

请在指定路径新建文件夹（英文名），右键空白处，选择 Git Bash here，输入指令：

hexo init

安装主题

安装你喜欢的主题 打开安装文件夹下_config.yml 文件，修改 theme 后面的名字为你刚才安装的主题

清除 hexo

hexo clean

部署 hexo

hexo g

启动 hexo 本地化预览

hexo s

hexo git推送

hexo d

Linux服务器部署方式

nginx yum安装

yum -y install epel-release 
yum -y install nginx
nginx默认基础文件地址 /etc/nginx/
html默认存放地址 /usr/share/nginx/html/

hexo g 命令后把静态文件存放即可

centos关闭防火墙

1、查看防火墙状态
firewall-cmd --state
2、停止firewall
systemctl stop firewalld.service
3、禁止firewall开机启动
systemctl disable firewalld.service 
4、新增开放端口
firewall-cmd --zone=public --add-port=端口号/tcp --permanent
5、移除开放端口
firewall-cmd --zone=public --remove-port=端口号/tcp --permanent
6、查看开放的端口
firewall-cmd --zone=public --list-ports
7、刷新防火墙
firewall-cmd --reload

Github page部署方式

ssh公钥生成

$ ssh-keygen -o

生成路径 C:\Users\用户名\.ssh

hexo 准备工作

编辑_config.yml文件 配置repo地址

hexo

安装 hexo github page 部署插件

npm install hexo-deployer-git --save

进行部署上传

hexo clean
hexo g -d

第三步

在Github中，找到托管博客的xxx.github.io项目，进入到设置页面
第四步
